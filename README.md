# FORO-HUB-APIREST ANGULAR Y SPRINGBOOT


##  SOBRE EL CODIGO 
Desarrollo basado en los criterios del Challengs 3 Creacion FORO HUB, un aplicativo que predende crear interacion entre usuarios proponiendo preguntas o temas para que posteriormente otros usuarios puedan brindar informaci贸n en el desarrollo se implementan de 3 clases base USUARIOS, TEMAS, RESPUESTAS con funciones CRUD, subida de imagnes perfil, relacion entre tablas para consulta de datos en un API REST que puede ser consumida por un FRONTED.

-  FILTRAR ID
-  ELIMINACION ID
-  CRUD USUARIO, TEMA, RESPUESTA
-  SUBIDA DE ARCHIVOS
